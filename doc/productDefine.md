# 产品定义 

## 名称
基于区块链的预制菜生产监测和溯源系统

<br/>  

## 发布时间		
2023年11月3日下午
<br/>

## 目标客户		
* 广大食用预制菜的消费者
* 以及生产预制菜的厂商
* 销售企业和监管部门

<br/>

## 解决了什么问题	
* 监测预制菜生产过程，监测异常情况并发出警报且及时挽救
* 利用区块链的不可篡改性和透明性，使得产品的各种生产数据对顾客完全公开，让顾客吃得安心

<br/>

## 对客户的价值
* 透明公开预制菜来源，使消费者放心。
* 通过品牌信任提供可信平台，使制作厂商有第三方公证平台可依赖。
* 为销售公司提供商品质量保证，寻找可靠制作商。
* 为监管部门提供追责依据，更快查询食品来源问题。

<br/>

## 解决方案
* 制作Web应用程序

<br/>

## 差异化方法
* 区块链避免生产信息造假且生产信息公开透明。
* 为消费者提供快捷食品安全检举平台。
* 提供生产监测平台。
* 提高自动化水平，减少人力消耗。

<br/>

## 产品实现的主要里程碑
* 智能合约开发完成
* 数据成功上链
* 初次客户体验使用并反馈测试感受
* 初步投入使用
<br/>
